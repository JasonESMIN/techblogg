Class Selectors(BEM)

See the Pen Class Selectors (BEM) by Captain-Absol (@captain-absol)on CodePen.

A block is a top-level abstraction of a new component like a button/.btn, this can be thought of as a parent. Elements can be thought of as child items, they can be placed inside blocks and are denoted by two underscores following the name like .btn__price. Modifiers can manipulate the block to give a theme or style to a specific component without inflicting changes on a completely unrelated module, they can be denoted with two dashes following the name like .btn--orange.
